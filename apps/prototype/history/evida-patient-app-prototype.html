State coverage

✔ Happy path ⛔ Error / failure 📭 Empty / partial 🔒 Auth gate ⏳ Loading 📌 Edge case

Key edge cases covered

• GP slot gating (4–5 working day window with plain-English reason)
• 15-min slot hold countdown during checkout
• Payment failure recovery (slots still held)
• Slot expiry (released → pick new)
• Locked account (too many attempts)
• Empty/partial: no wearables, no reports, no appointments, incomplete onboarding
• Coming-later options visible with cost context (nurse visit, home kit)
• Non-blocking pre-consult tasks with skip/connect-later
• ID verification skip (defer to later)
• Fasting/preparation reminder auto-offered
