# Evida

**Know Your Health. Know What to Do.**

Evida is a GP-led health platform that combines your data, technology, and clinical expertise into a comprehensive health check, helping you take action for a healthier you.

## The Evida Loop

- **Track** — Connect your wearable, lab tests, and medical history into one secure record
- **Tailor** — AI-supported analysis + clinical expertise provides personalised insights
- **Act** — A dedicated 45-min consultation with actionable next steps

## Tech Stack

- **Framework:** Next.js 16
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4
- **Animation:** GSAP
- **Build Tool:** Turbopack

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the site.

## Available Scripts

| Command          | Description                  |
| ---------------- | ---------------------------- |
| `npm run dev`    | Start dev server (Turbopack) |
| `npm run build`  | Build for production         |
| `npm run start`  | Start production server      |
| `npm run lint`   | Run ESLint                   |
